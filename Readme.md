Glowplus
製作者：Panko200
配布場所：https://github.com/panko200/Glowplus

## 概要

いい感じの光を出すエフェクトが欲しかったので作りました。いい感じの光を出します。

## 使用方法

映像エフェクト「Glow plus」が追加されます。  
標準の「発光」(bloom)よりも、綺麗に発光すると思います。きれいであってほしい。Glow plus、キレイ？これでも？

## アンインストール方法

1. YMM4 を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4 を終了する。
3. `GlowplusEffect`という名前のフォルダを削除する。

## 注意点

OS : Windows11 (64bit)  
ゆっくりMovieMaker4 : v4.51.0.3  
にて動作確認をしています。

**作者は、本プラグインの使用または使用不能に起因するいかなる損害についても、一切の責任を負いません。**

バウンディングボックス(当たり判定)がクソほどでかくなります。普通にえぐいほどでかくなります。クッソ編集の邪魔になるので、一回設定終わったらそのレイヤー(もしくはアイテム)を非表示にしておきましょう。

なんでかわからないけど品質を下げれば下げるほど重いです。10 のまま使うことをお勧めします。

思ったよりも RAM を使います。適応する画像のカメラ位置を 0 に近くすると、バウンディングボックス(当たり判定)が
とんでもなく大きくなるので、その時は結構 RAM を使ってしまいます。が、一応、それの対処法として、「自動クリッピング」
というオプションを追加しております。自動クリッピングを OFF にしているときは、上記の行動をするとエグ RAM 使うので、気を付けましょう！
